# Changelog

All notable changes to Tova will be documented in this file.

Format based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [Unreleased]

### Added
- DOOM-style pixelation post-process shader for retro chunky visuals
- View bobbing while moving (subtle head sway effect)
- First-person weapon viewmodel with movement bob

## [0.1.0] - 2026-01-25

### Added
- First-person player controls with pointer lock
- Procedural terrain with height mapping
- Castle structure (keep, towers, walls)
- Procedural town with houses
- Animated ocean
- Environment lighting, fog, and sky
- Fly/walk mode toggle via chat commands (`/` to open chat, type `fly` or `walk`)
- Postprocessing pipeline with bloom effects
